## 2026-03-12 - [Feedback for Destructive or Success Actions]
**Learning:** Users need immediate, clear visual feedback when an action (like copying text or deleting an item) is successful. Relying only on the state change of the data is often too subtle.
**Action:** Use color transitions (e.g., blue to green) and icon changes (e.g., adding a checkmark) to confirm success, and ensure these states are accessible to screen readers via ARIA labels or live regions.
